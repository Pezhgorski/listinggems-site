Changelog

Release notes and version history for ListingGems.

v0.2.1

May 23, 2026
  • Color Select preview now matches what Apply will do. The overlay shows exactly which pixels will be affected, including ones outside the current foreground. Restore can highlight matching colors anywhere in the photo.
  • When Restore is active, the canvas swaps to show the full original image with a dim tint over removed areas, so you can see what is about to come back into the mask. Tolerance scrubbing stays instant.
  • Erase keeps the regular product display to focus on what is being subtracted — no distraction from background pixels.
  • Flood fill no longer treats the existing mask edge as a wall, so a color region can be reached even when it spans across the foreground.

v0.2.0

May 21, 2026
  • Mask brush now supports zoom and pan for pixel-precise edge work: Ctrl+scroll to zoom, middle-mouse drag to pan, and keyboard shortcuts (0 to reset, + / - to zoom). A live cursor coordinate readout helps you align edits.
  • Fixed brush erase and restore coordinates so paint lands exactly under the cursor when working on a cropped preview.
  • Brush mode menu renamed for clarity: Brush (previously Smart) and Color Select, with Brush as the new default.
  • Added a dismissable hint banner explaining the brush in both Studio and Quick modes.
  • Background removal model picker now hides models your GPU can't run (FP16 support detection), so you only see options that will work on your hardware.
  • HEIC files that are actually JPEGs (such as Samsung motion photos) now load correctly instead of failing.

v0.1.2

May 15, 2026
  • Background removal models now preload in the background after app startup, so the first photo you process no longer waits on model initialization.

v0.1.0 — Initial release

April 2026
  • Background removal powered by BiRefNet and BEN2 ONNX models, running locally on GPU or CPU — no cloud, no per-image fees.
  • Studio Mode for fine-grained editing: drop shadows, color and exposure correction, smart 4x upscaling, and a manual mask touch-up brush for refining edges.
  • Quick Mode for drag-and-drop batch processing with progress and remaining-time estimates.
  • Nine marketplace presets covering eBay, Etsy, Amazon, Shopify, Poshmark, Mercari, Depop, Facebook Marketplace, and a custom aspect ratio.
  • One-time purchase of $49 with no subscription, three device activations per license, and a 45-day money-back guarantee.
  • Runs fully offline after install — photos never leave your machine.
