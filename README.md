# ListingGems — Landing Page

Source for **[listinggems.com](https://listinggems.com)**, the marketing site for ListingGems.

## About ListingGems

ListingGems is a desktop product photo enhancer for marketplace sellers. Drop in your photos and get professional-looking listings in seconds — clean backgrounds, soft drop shadows, color correction, smart cropping for every marketplace.

- One-time purchase, no subscription
- Runs fully offline on your own machine — photos never leave your computer
- Built for eBay, Etsy, Poshmark, Amazon, Shopify, Mercari, Depop, and Facebook Marketplace sellers
- Uses on-device ML segmentation (BiRefNet / BEN2), not generative AI — your products stay your products

Learn more or download at **[listinggems.com](https://listinggems.com)**.
